# ManyQ

### Usage?
Some multi thread query tool with vpn to bypass ratelimit  
Config the json and run with `python3 main.py --config config.json`  
The example json config is included  
You will need firexfox webdriver, install with `apt install firefox-geckodriver` or other package manager
If you want to config the vpn service, modify the `vpnswitch.py` and add your own cmd
